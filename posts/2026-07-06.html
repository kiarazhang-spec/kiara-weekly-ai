🎯 本周 Top 5

01.OpenAI 同日发布 GPT-5.6 三档 + ChatGPT Work：企业 Agent 从概念被打包成付费标品

来源：OpenAI 官博 · GPT-5.6（2026-07-09）；OpenAI 官博 · ChatGPT Work（2026-07-09）

GPT-5.6 正式 GA，分 Sol / Terra / Luna 三档，API 定价分别 $5/$30、$2.50/$15、$1/$6（每百万 token 输入 / 输出），Sol 同步成为 ChatGPT 默认模型，也是 Microsoft 365 Copilot 的首选模型。同日发布的 ChatGPT Work 是内置 Codex 能力的企业 Agent，支持连接 1400+ 工具、跨应用自动拆解目标、生成表格 / 幻灯片 / 网站等"成品交付物"，附带 Plan 审批、审计日志与合规 API。Pro / Enterprise / Edu 首批开通，Plus / Business 后续滚出；Codex 桌面端并入统一 ChatGPT 客户端。

产品技术视角：OpenAI 一天内做了两件事——用三档定价卡住"低成本高频"（Luna 直接对标 Sonnet），再用 ChatGPT Work 把 Agent 从演示品变成企业采购目录里的一行。真正值得注意的信号是产品叙事从"写代码"扩到"完成工作"：Chat / Work / Codex 三个入口平级，"AI 员工"被正式做成了 SKU，直接和 Microsoft Copilot Studio、Anthropic Claude Code / Managed Agents 抢企业席位。

02.Anthropic 发布 GRAM：把双用途知识装进可拆卸模块，安全从"拒答"进到"权重级"

来源：Anthropic Research · An off switch for dual-use knowledge（2026-07-08）

Anthropic 与 AE Studio 合作发布论文 GRAM（Gradient-Routed Auxiliary Modules），在 Transformer 每层加入按风险类别分组的辅助模块——训练时遇到病毒学 / 网络安全 / 核物理等敏感数据，梯度只写入对应模块，通用权重冻结；训练完成后可整段删除该模块，"能力随权重一起被拿掉"。实验覆盖 4 个风险类别、50M–5B 参数 7 档模型规模，删除模块的抗微调恢复能力接近"从未训练过该数据"，且在前沿规模下开关模块的效果差距随模型规模增大而更明显。Anthropic 明确说该方法尚未用于生产模型。

产品技术视角：这是一次少见的"结构性安全"进展。以前所有安全手段都在模型学会之后做拦截（拒答训练 / 分类器 / 越狱防御），GRAM 是把知识本身路由到可拆解的地方。对企业 / 政府客户意义很大：一次训练就可以按"这个客户可以看病毒学、那个客户不行"发不同权重版本，而不需要每次重训。这也把 Anthropic "安全 = 差异化"的战略推到新的技术底层——未来出口管制、生物安全审查大概率会向这个方向靠。

03.Microsoft 自研 MAI 模型在 Excel / Outlook 每周处理数万 prompt：向 OpenAI × Anthropic 依赖开始拆解

来源：Bloomberg（2026-07-07）；The Information（2026-07-08）

Bloomberg 报道，微软已开始在 Excel、Outlook 中用自研 MAI 系列替代 OpenAI / Anthropic 模型，两款办公套件每周有数万条 AI prompt由 MAI 完成，此前从未公开披露。微软 AI 负责人 Mustafa Suleyman 直接表态："我们向 Anthropic 支付大量资金，目标是减少并最终消除这部分成本。" Bloomberg 引述内部数据指出 2026 年初微软向 Anthropic 支出约 5 亿美元 / 年。MAI 模型体系（6 月 Build 大会一次发 7 款）已接入 GitHub Copilot，自研语音转写将进入 Teams。

产品技术视角：2025 年那次 Microsoft × OpenAI 合作条款修订（去除微软自建前沿模型限制）到今天开始收效——曾经把 AI 战略绑在 OpenAI 上的巨头，现在明确要"降低 / 最终消除"外部模型支出。这条新闻对整个 AI 产业链的定价意义远大于对办公产品的意义：如果连微软都开始在高频场景切自研模型，OpenAI / Anthropic 的企业单价长期天花板就已经被划定。模型层利润率的顶部拐点，可能已经出现。

04.Meta 发布 Muse Spark 1.1：史上第一次给自家旗舰模型收费，用 $1.25 / $4.25 直接砸市场

来源：Meta AI · Muse Spark 1.1 Evaluation Report（2026-07-09）；Bloomberg（2026-07-09）

Meta Superintelligence Labs 发布 Muse Spark 1.1，是 Meta 有史以来第一次给自家旗舰模型标价，取代此前 Llama 系列的开源免费策略。定价 $1.25 / $4.25 每百万 token（输入 / 输出），约为 GPT-5.6 Terra 输出价的 28%、Claude Opus 4.8 输出价的 17%。首日仅在美国公开预览，新用户 $20 免费额度，暂不上 OpenRouter 等第三方平台。技术侧主打 100 万 token 上下文、原生多模态（文本 / 图像 / 视频 / 计算机使用）、端到端 Agent 与仓库级编程。Zuckerberg 时隔三年重回 X 亲自发布。

产品技术视角：Meta 的转向意味着"免费开源前沿模型"时代基本结束。Muse Spark 1.1 是 Alexandr Wang 领导下 MSL 的产物，走的是闭源 + 低价 + 高频调用路径，不是打质量而是打价格战——把定价放在 OpenAI 的四分之一，本质是逼 OpenAI / Anthropic 降价。这一周同时叠加微软 MAI，两条独立的新闻其实是同一个信号：Token 价格战正式开打，边际利润率被结构性压低。

05.Apple 起诉 OpenAI + io Products：40 页诉状指控系统性窃取硬件商密，AI 战线首次烧到硬件供应链

来源：Bloomberg（2026-07-10）；TechCrunch（2026-07-10）

Apple 在加州北区联邦法院（案号 5:26-cv-07078）起诉 OpenAI Foundation、OpenAI Group PBC、io Products 及两名前苹果员工，包括现任 OpenAI 首席硬件官 Tang Yew Tan（前 Apple iPhone / Watch 产品设计副总）与工程师 Chang Liu。诉状 40 页、含具体证据链：Chang Liu 离职后利用 Apple 内网身份验证漏洞下载未发布产品文件，并邮件写道"哈哈，我发现我能访问网络存储"；诉状披露已有超 400 名前 Apple 员工加入 OpenAI，苹果曾于 2 月致函 OpenAI 未获回复。Apple 请求全面禁令、返还销毁涉密资料，甚至要求法院强制 OpenAI 重新设计其硬件产品剔除 Apple 专利技术。OpenAI 回应"对其他公司商业秘密不感兴趣"。

产品技术视角：这次诉讼真正的意义不是"能不能赢"，而是AI 竞争的战场正式从模型层扩展到消费硬件供应链。此前 Apple × OpenAI 是合作方（Siri 接入），而今在 OpenAI 上市前夜（IPO 敏感期）把矛盾变成公开法律事件。诉状里"OpenAI 硬件业务根基不稳，其核心早已因非法窃取而腐朽不堪"的措辞极其少见——Apple 的目标不是赔偿，是在监管与投资人层面给 OpenAI 的硬件路线打上"合规风险"标签。同期传出 Siri 转向 Google Gemini，两件事叠加，是 Apple 对 OpenAI 的战略切割。

🔍 本周三大主题

主题一：Agent 从"能演示"迈进"可采购"——企业席位战开打

  • OpenAI 官博（2026-07-09）：ChatGPT Work 由 GPT-5.6 + Codex 驱动，跨 1400+ 应用自动拆解目标，交付表格 / 幻灯片 / 网站，配 admin 审计与合规 API。
  • OpenAI × Microsoft（2026-07-09）：GPT-5.6 成为 Microsoft 365 Copilot 首选模型，Sol 直接进企业办公链。
  • Anthropic Managed Agents（2026-07-06）：Managed Agents 加入事件 delta 流、单会话 override、全生命周期 webhooks；Claude Code 默认权限模式改为 Manual（更保守的自主性默认值）。
趋势研判：三家一齐把 Agent 做成"企业采购单位"——OpenAI 走 Chat / Work / Codex 三合一入口、Anthropic 深耕开发者侧 Managed Agents、Microsoft 用 Copilot 抓办公套件。真正的差异化在"审批链 + 审计 + 撤销"三件套：Chat Work 的 Plan 审批、Claude Code 的 Manual 默认、Copilot 的 admin 控制，是这一波企业 Agent 落地的通用基线。对 PM 意义：做 AI 企业产品的默认合规栈已经形成，不再是可选功能。

主题二：Token 价格战正式开打——两条独立路径同一周指向"边际利润被压低"

  • Bloomberg（2026-07-07）：微软 MAI 模型在 Excel / Outlook 每周处理数万 prompt，Suleyman 明言"最终消除"对 Anthropic 支出，年支出约 5 亿美元。
  • Meta AI（2026-07-09）：Muse Spark 1.1 首次付费定价 $1.25 / $4.25，为 GPT-5.6 Terra 输出价 28%、Claude Opus 4.8 输出价 17%。
  • OpenAI（2026-07-09）：GPT-5.6 Luna 定价 $1 / $6，明显对标 Sonnet 与 Meta 新品，同档最激进。
趋势研判：本周同时出现两条独立信号——大买家（Microsoft）不再愿付溢价、新供给方（Meta）主动打半价——共同把模型层利润率天花板明确划下来。未来 12 个月要观察的不是"谁的模型更强"，而是"谁能在低毛利下养住研究团队"。这也解释了为什么 OpenAI 要抢做 ChatGPT Work、Anthropic 要押 Managed Agents / Claude Code——都是把利润从 API 单价迁移到"应用层订阅 + 企业席位"。

主题三：AI 竞争第一次从模型层扩展到硬件供应链与安全权重结构

  • Bloomberg（2026-07-10）：Apple v. OpenAI 40 页诉状，400+ 前 Apple 员工跳槽 OpenAI，Tang / Liu 被点名，请求法院强制重设计 OpenAI 硬件。
  • Anthropic Research（2026-07-08）：GRAM 把危险知识路由到可拆模块，一次训练可衍生 16 种能力组合的权重版本。
  • TechTimes / 综合报道（2026-07-11）：Siri 传转向 Google Gemini 底座，Apple 与 OpenAI 从合作转向直接竞争。
趋势研判：过去 3 年 AI 竞争主战场是模型能力，现在同一周同时向两侧扩：硬件（Apple × OpenAI 诉讼）与权重结构（Anthropic GRAM）。这标志着 AI 正在被"资产化 + 结构化 + 主权化"——硬件商密 = 供应链主权，权重可拆模块 = 出口管制的技术底座。下一阶段的 AI 政策与并购逻辑，将围绕这两条线展开，而不是单一模型评测榜。
