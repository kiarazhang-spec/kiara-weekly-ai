🎯 本周 Top 5

01.Anthropic 发布 Claude Sonnet 5 与 Claude Science：中端模型塞进 Opus 级推理，同步推出"科研 Agent"

来源：Anthropic 官博 · Sonnet 5（2026-06-30）；Anthropic 官博 · Claude Science（2026-06-30）

Anthropic 在同一天发布 Sonnet 5 与配套产品 Claude Science。Sonnet 5 保持 Sonnet 定价的前提下，把 ARC-AGI-2、SWE-bench Verified、GPQA 三项跑分推到接近上一代 Opus 的水平；Claude Science 是面向研究人员的 Agent，能读多篇 PDF、跑 Python 数据分析、并交叉引用文献做假设推演，官博单独强调它已在生物、材料、气候三个领域接入外部实验室试点。

产品技术视角：Anthropic 这一步的定价策略比模型本身更值得看：Sonnet 5 的目标不是打 GPT-5.6，是打"你为什么还要花 Opus 的钱"。同时 Claude Science 是继 Claude Code 之后第二个"垂直 Agent + 特定 workspace"打包产品——Anthropic 正在把"通用 Chat + 垂直 Agent"确立成自己的产品组合方式。

02.OpenAI 发布 GeneBench-Pro：GPT-5.6 Sol 只拿到 31.5%，第一次给"AI 做生物学"划出人类基线

来源：OpenAI 官博（2026-06-30）

GeneBench-Pro 由 OpenAI 联合 42 位分子生物学家出题，覆盖分子机制推理、实验设计、湿实验故障排查等 8 个子任务，明确要求模型给出"可复现的实验流程 + 预期结果 + 排错策略"。GPT-5.6 Sol（Thinking 版）总分 31.5%，人类博士基线约 68%；Claude Sonnet 5 与 Gemini 3 Ultra Reasoning 分别为 27% 与 24%。OpenAI 首次在评估里加入"实验被质疑时能否合理反驳"这一维度。

产品技术视角：把 31.5% 大方公布，正好卡在"不算强、但不是零"的位置，OpenAI 是主动为下一版模型留下叙事空间。真正的信号是评估内容——从"能答题"变成"能设计实验并复盘"，这暗示 GPT-6 世代要打的是"AI 作为科研合作者"，而不是"AI 作为答题机器"。

03.Meta 官宣 Llama API 于 7 月 6 日关停：开源阵营正式退出托管市场

来源：Meta AI 官博（2026-07-06）；TechCrunch（2026-07-06）

Meta 宣布关停 Llama API 托管服务，理由是"生态里的第三方托管商（Groq、Together AI、Fireworks 等）已经覆盖绝大多数场景"。同时确认 Llama 4 Behemoth 系列继续以权重开源方式发布，未来 Meta 自身不再运营对外收费的推理端点。Meta AI 消费级产品（Meta AI app、Ray-Ban Display AI 眼镜内嵌）则继续使用内部 Llama 推理集群。

产品技术视角：这次撤退是清晰的战略聚焦——Meta 承认自己不做"卖 token"的生意，把这一段让给专攻 inference 的 Together / Groq / Fireworks。真正的资金和精力将集中到硬件（AI 眼镜）+ 消费级 App。对开源生态是好消息：Llama 权重继续开放；对企业买家则要留意——想用 Llama 就必须走第三方托管商，采购路径变了。

04.OpenAI 提议向美国政府发行 5% 股权：主权 AI 从叙事推进到公司治理层

来源：Financial Times（2026-07-03）

FT 引述多位知情人士称，OpenAI 已向美国政府提交一项"5% 股权换战略级基础设施合作"提案，条款包括：政府获得约 5% 无投票权股份、在国家安全事项上有一票否决权、OpenAI 承诺美国境内新建算力份额不低于总量 60%。方案与 Stargate 项目挂钩，但独立于 Stargate 单独定价。白宫暂未表态，OpenAI 官方回应称"我们与政府在多个层面持续沟通，具体条款不予置评"。

产品技术视角：如果条款落地，这将是自二战 RCA / Bell Labs 以来第一次美国政府用股权方式介入一家私营科技公司。对行业的连锁反应会集中在两点：Anthropic / xAI 是否会跟进类似安排；以及"主权 AI"从此不再是营销话术，而是资本表内的一栏。

05.Google DeepMind 发布 Nano Banana 2 Lite + Gemini Omni Flash：多模态"低价 + 全模态"双管齐下

来源：Google DeepMind 官博（2026-07-01）

Nano Banana 2 Lite 把图像生成 / 编辑价格压到每张 0.008 美元，官博点名"看齐即梦 4o Mini 的开发者定价"；Gemini Omni Flash 首次把文本、图像、音频、视频、代码执行在一次 API 调用里打通，官博演示"给一段 3 分钟视频，输出配音 + 中英字幕 + 关键帧图片 + JSON 事件日志"。两者共用同一个新模态融合骨干，DeepMind 明确说下一步会把这个骨干下沉到 Gemini 3 Ultra。

产品技术视角：这是 Google 少见地把"打价格"和"打技术"放在同一天讲。Nano Banana 2 Lite 明显在对齐国内即梦 / 可灵的 API 单价——过去一年国内多模态 API 单价一直低于海外 3-5 倍，这次是 Google 首次贴身近战。Omni Flash 则把"多次调用 + 拼接"这类中间层生意从此挤压出去，Zapier / n8n 里做"视频 → 字幕 → 图片 → JSON"的中间件将首当其冲。

🔍 本周三大主题

主题一：评估战场从"对话基准"转向"科研 / 生物 / 工程"三个专业赛道

  • OpenAI 官博（2026-06-30）：GeneBench-Pro 由 42 位分子生物学家出题，人类基线 68%，GPT-5.6 Sol 31.5%。
  • Anthropic 官博（2026-06-30）：Claude Science 与生物、材料、气候三个领域外部实验室试点。
  • Anthropic 官博（2026-06-30）：Sonnet 5 在 ARC-AGI-2、SWE-bench Verified、GPQA 三项对齐前代 Opus 水平。
趋势研判：MMLU / GPQA 已经饱和，实验室竞争必然向"具备湿实验或工程闭环的领域"迁移。这也解释了为什么 OpenAI 和 Anthropic 都在同一周把生物学抬出来做主推——生物学有清晰的 ground truth（实验能不能复现），比法律、金融更适合作为下一个"能量化的圣杯"。

主题二：开源阵营的商业化路径收缩到"权重 + 硬件"，让出"托管服务"

  • Meta AI 官博（2026-07-06）：Llama API 于 7 月 6 日关停，托管让给 Groq / Together / Fireworks。
  • Together AI 官博（2026-07-01）：完成 8 亿美元 C 轮，估值 83 亿美元，明确定位为"开源模型的 AWS"。
  • TechCrunch（2026-07-06）：Meta 内部 Llama 推理集群继续服务 Meta AI app 与 Ray-Ban Display。
趋势研判："开源阵营"不再是单一角色，正在分层——Meta 做权重 + 硬件平台、Together / Groq / Fireworks 做托管、企业用户做微调。未来 12 个月值得关注："谁能拿到 Llama 4 Behemoth 的独家企业级 fine-tune 授权"——这将决定第二梯队谁能坐稳"开源生态里的 Databricks"位置。

主题三：主权 AI 从政治叙事进入公司治理，同时中国资本继续用估值抢占叙事高地

  • Financial Times（2026-07-03）：OpenAI 提议向美国政府发行 5% 无投票权股份 + 国安一票否决。
  • OpenAI 官博（2026-06-28）：OpenAI × HP Frontier PC 合作，把 ChatGPT Business 内嵌进企业级 Windows 采购链。
  • Kling AI 官博（2026-07-02，引 FT / 36Kr 报道）：Kling AI 完成约 30 亿美元 C 轮，估值 180 亿美元，成为国内视频模型第一名。
趋势研判：美国这边"政府股权 + 国产算力承诺"和"OpenAI × HP 的桌面渗透"是同一件事的两面——把 AI 供应链绑进本土采购与国安审查。中国这边则用估值 + 产业资本推 Kling / Kimi / 昆仑心继续占据模态与算力叙事。两条路径的共同点是：AI 已经从纯技术产品，被彻底"资产化 + 主权化"。
