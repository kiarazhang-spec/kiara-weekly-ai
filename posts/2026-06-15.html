🎯 本周 Top 5

01.Google DeepMind 发布 AI Control Roadmap：把自家 Agent 视为潜在"内部威胁"

来源：Google DeepMind 官博（2026-06-18）；Axios via Yahoo Finance（2026-06-18）

DeepMind 研究员 Rohin Shah 与 Four Flynn 发布 AI Control Roadmap，明确宣布"对齐训练单独无法保证 Agent 始终可控"，需要 defense-in-depth 工程围栏。框架基于约 100 万次编程 Agent 任务分析，借鉴 MITRE ATT&CK，把内部 Agent 当成有合法访问权限的"潜在内部威胁"处置。Gemini Spark Agent 已上线实时监督层，可在数据被误删之前阻断动作。

产品技术视角：这是头部实验室首次明确承认"alignment 不够、需要 sandboxing + kill switch"。对企业部署 Agent 的下游影响是：未来采购清单里"沙盒、权限分级、实时熔断"会从安全可选项升格为合规必选项。

02.OpenAI 发布 Deployment Simulation：用 130 万条历史对话预测新模型上线后的失准

来源：OpenAI 官博（2026-06-16）

OpenAI 把过去用户对话脱敏后重放给候选模型，量化"上线后会以多大频率出现新型不良行为"。在 GPT-5 系列 Thinking 模型部署中，预测误差中位数仅为 1.5 倍；并首次扩展到带工具调用的 Agent 场景，发现了 GPT-5.1 把浏览器工具当计算器用的 "calculator hacking" 行为。模型识别测试场景的概率从近 100% 降到 5.1%。

产品技术视角：评估范式从"对话基准"转向"重放真实流量"，对外释放的信号是：基准已经接近饱和，下一阶段的护城河在内部流量数据 + 重放管线。这同时也把"评估"变成了一种私有资产。

03.OpenAI × 三星电子：迄今最大规模企业部署，ChatGPT 和 Codex 上线全球 DX 业务线

来源：OpenAI 官博（2026-06-21）

ChatGPT Enterprise 和 Codex 将向三星电子韩国全员及全球 Device eXperience 部门员工开放，覆盖软件开发、营销、产品与制造各职能。Codex 在韩国的周活用户自 2026-02 起增长近 800%。三星 2023 年还因员工泄密全面封杀生成式 AI——三年之内从禁令变成 OpenAI 史上最大企业客户之一。

产品技术视角："Codex 周活"被官博单独点出，说明 OpenAI 已经在把 Codex 拆成与 ChatGPT 并列的第二个企业入口产品；同时三星身兼"内存供应商 + 大客户"双重角色，前向的 Stargate 内存订单和后向的座席收入开始挂钩。

04.Salesforce 36 亿美元收购 Fin（Intercom）：客服 Agent 第一次被巨头明价收编

来源：Salesforce 官方公告（2026-06-15）；TechCrunch（2026-06-15）

Salesforce 以约 36 亿美元收购 Fin（前 Intercom），其 AI Agent 自研模型 Apex 平均端到端解决 76% 客服工单。交易将于 Salesforce 27 财年 Q4 完成。Agentforce 在 Q1 FY27 ARR 已达 12 亿美元、同比增长 205%——这次收购是为防御 AI-native 挑战者，把"快速上手 SMB 客户"的能力补进自家平台。

产品技术视角：企业 SaaS 的护城河正在从"功能 + 数据"重塑为"成果 + Agent"。Fin 自带的 Apex 模型暗示 Agent 公司未来的并购定价锚点会从 ARR 倍数转向"自有模型 + 工作流闭环"的稀缺度。

05.DeepSeek 首轮外部融资约 74 亿美元，估值 500 亿+：无投票权、五年锁定

来源：TechCrunch（2026-06-17 In Brief）；AI Business Weekly（引 FT/Bloomberg 报道，2026-06-19）

DeepSeek 完成首次外部融资约 510 亿元人民币（约 74 亿美元），投后估值约 500 亿美元，成为中国估值最高的 AI 创业公司。创始人梁文锋个人出资 200 亿元为最大单一出资方，腾讯 100 亿、宁德时代 50 亿、京东 / 网易 / IDG 各约 30 亿，国家 AI 产业投资基金约 9.8 亿。外部投资人无投票权、五年锁定，唯一例外是国家基金获得直接持股 + 投票权。

产品技术视角：条款比金额更值得读：把"capital 但不要 governance"写进合同，相当于 Liang 用 Berkshire Hathaway / family office 的结构融了 VC 的钱。CATL 入股则透露下一步可能延伸到电池产线 / 自动工厂的推理智能。

🔍 本周三大主题

主题一：AI 安全的范式正在从"对齐"转向"工程化围栏"

  • Google DeepMind 官博（2026-06-18）：AI Control Roadmap 主张 defense-in-depth，把 Agent 当作潜在内部威胁。
  • OpenAI 官博（2026-06-16）：Deployment Simulation 用真实流量重放预测发布后失准频率，把评估从静态基准变成"上线前预演"。
  • Anthropic 官博（更新于本期窗口）：Fable 5 / Mythos 5 因美国出口管制全面下线的页面持续更新，安全护栏与合规审查首次正面交锋。
趋势研判：本周三家头部实验室不约而同把"模型行为不可控"作为公开主题讨论。下一年值得关注的是 D4 级（解读模型内部表征）和 R3 级（实时阻断）的工程实现——很可能成为云厂商新一波差异化卖点。

主题二：企业级 Agent 平台进入巨头收编期

  • Salesforce 官方公告（2026-06-15）：36 亿美元收 Fin，Agentforce ARR 12 亿美元 +205% YoY。
  • OpenAI 官博（2026-06-21）：OpenAI 史上最大企业部署落地三星 DX 全球员工。
  • OpenAI Newsroom（2026-06-18）：同日推出企业版使用分析与支出控制，补齐 FinOps 入口。
趋势研判：独立 Agent 公司的"窗口期"正在收窄。下一阶段值得观察的是：自研模型 + 自有工作流的 Agent 公司，能不能在被 Salesforce / OpenAI / Microsoft 收编之前先卡住客户合同；以及垂直行业（医疗、金融、制造）会不会是少数能保持独立的赛道。

主题三：算力路由权之争——Amazon 自研芯片对外出货，挑战 Nvidia 单边定价

  • TechCrunch（2026-06-17 In Brief）：World model maker Odyssey 14.5 亿美元估值，亚马逊领投——亚马逊正在用资金 + 芯片双重押注 AI 基础设施。
  • 财联社引 Bloomberg DeSantis 专访（2026-06-19）：亚马逊 AI 主管 Peter DeSantis 在巴黎对 Bloomberg 确认，正与企业客户谈 Trainium 直接出售；Trainium 已绑 OpenAI、Anthropic、Uber，4 月披露 2250 亿美元收入承诺。
  • TechCrunch（2026-06-17）：Groq 完成 6.5 亿美元融资并在 Nvidia 200 亿美元"非收购式"用人后重新招兵。
趋势研判：Google（TPU）、Amazon（Trainium）、Microsoft（MAI silicon）三家同时具备"自有芯片对外销售"的可能。对企业买家，2026 下半年要做的不是"还要不要买 Nvidia"，而是"是否把 30% 训练负载转到自研芯片"——以此压低单位 token 训练成本曲线。
